# MIDI Implementation Documentation

The official Korg MIDI Implementation documents for the electribe2 and electribe sampler are **not included** in this repository (copyright © KORG Inc., for personal non-commercial use only).

You can download them directly from Korg:

- **electribe2 synth**: https://www.korg.com/us/support/download/manual/0/367/2666/
- **electribe sampler**: https://www.korg.com/us/support/download/manual/0/368/2665/

Place the `.txt` files in this directory (`docs/`) if you want to work with the source code locally. The app does not require them to run — they are reference material for protocol understanding.
